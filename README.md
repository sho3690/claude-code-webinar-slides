# 非エンジニアでもここまでできる — Claude Codeで変わる日常業務

社内ウェビナー用のスライド資料です。

## 概要

エンジニアではない立場から、Claude Codeを使って日常業務を自動化した4つの事例を紹介します。

| # | 事例 | 内容 |
|---|------|------|
| 01 | 健康データの自動レポート化 | Oura Ring → AI分析 → Slack配信 |
| 02 | YouTube動画のAI要約 | 新着動画 → 字幕取得 → Claude要約 → Slack |
| 03 | 情報キュレーションの自動化 | RSS/Web/X → AI要約 → Weekly Digest |
| 04 | ワンショットのディープリサーチ | 自然言語指示 → 並列検索 → PDF(15-30p) |
